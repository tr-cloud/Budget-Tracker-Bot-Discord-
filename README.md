# Budget-Tracker-Bot-Discord-

## Overview

Budget Tracker Bot is a Discord bot developed in Java that helps users manage their personal finances directly within a Discord server. The bot allows users to record income and expenses, monitor spending habits, and view financial summaries through simple commands.

## Features

* Add and track income records
* Record daily expenses
* Monitor budget usage
* Generate financial summaries
* User-friendly command-based interaction
* Lightweight and easy to deploy on Discord servers

## Technologies Used

* Java
* Discord API (JDA)
* Replit
* Object-Oriented Programming (OOP)

## How It Works

1. Users interact with the bot using Discord commands.
2. Income and expense details are recorded by the bot.
3. The bot calculates totals and tracks spending.
4. Financial summaries are generated on request.

## Commands

| Command             | Description                |
| ------------------- | -------------------------- |
| `!income <amount>`  | Add income                 |
| `!expense <amount>` | Record expense             |
| `!summary`          | View financial summary     |
| `!budget`           | Check budget status        |
| `!help`             | Display available commands |
